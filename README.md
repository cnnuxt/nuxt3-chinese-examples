克隆源码：git clone https://github.com/eoctdm/nuxt3-chinese-examples.git   
（备选）：git clone https://gitee.com/eoctdm/nuxt3-chinese-examples.git  
进入工程：cd nuxt3-chinese-examples   
安装依赖：npm i   
运行项目：npm run dev   
浏览体验：http://localhost:3000
